> 安装依赖

- 访问 https://ollama.com 点击Download下载ollama客户端并安装
``` bash
pip install rich
pip install wxauto
pip install ollama
ollama run Qwen2:1.5b
```
### 注意，在`ollama run <moudel>`中，`<moudel>`为ollama的模型名称。不同的模型对GPU需求不一样，A、N卡均可
